变量

let 只在所在区块生效--先声明否则不可用--同一区块不可重复声明


const 声明常量--不可变--对象可变本身不可变


export 将变量暴露出去，可被其他文件请求获得

import 请求其他文件中的变量--写法：import * as var from './src'


window.var 全球变量

声明变量时不能用圆括号，只有赋值的时候才可以用圆括号 --如[(b)] = [3]; // 正确

yield 迭代器，遇到即终止返回值，如果继续请求则从终端处继续运行


解构可以赋默认值 var [foo =true] = [] foo//true
